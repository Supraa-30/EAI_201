# FIFA Pred Assignment 2
Project: Predictive modelling and exploratory analysis using FIFA data.
This folder contains the work for Assignment 2: exploratory data analysis, data collection/scraping utilities, model training experiments, and supporting scripts used to predict team performance / attributes from FIFA-related data.
## Contents
- `Codes/`
	- `EDA.ipynb` — exploratory data analysis notebook. Visualizations, feature inspection, missing-value checks, and initial preprocessing steps.
	- `MODELS.ipynb` — model-building notebook. Includes model training, evaluation, and comparison (baseline models and advanced approaches used in the assignment).
	- `scrapper.ipynb` — (note: spelled "scrapper" here) notebook used to collect or scrape data used in the analysis.
	- `Teams.py` — utility script for team-level data processing (feature engineering, mapping, helper functions).
- `Excels/` — (optional) exported datasets or results in spreadsheet form.
- `Reports/` — generated reports, images, or written analysis deliverables.
